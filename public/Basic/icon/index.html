Icon 图标

通用参数

所有组件均支持class和style属性，class可以是vue变量也可以是字符串
注意：变量需要在class前面加冒号（：）， 示例
<Icon class="ion-ionic" style="font-size: 12px;"></Icon>

概述

icon 图标使用开源的字体库ionicons

如何使用

使用Icon组件只需要指定图标的class即可，示例代码

<Icon class="ion-ionic"></Icon>

渲染后为

<i class="mv-icon ion-ionic"></i>

API

Icon props

属性 说明 类型 默认值 可选值
class 选填，类名（可以设置icon） String - 任意icon类
style 选填，内联样式,自由设置样式 [String,Object] - 任意
